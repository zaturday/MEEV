# MEEV: คลังข้อมูลและระบบวิเคราะห์แนวโน้มยานยนต์ไฟฟ้า (EV) ในประเทศไทย

ยินดีต้อนรับสู่ **MEEV** ระบบข้อมูล ข่าวสาร และระบบวิเคราะห์แนวโน้มของยานยนต์ไฟฟ้า (EV) ยอดนิยมในไทย ซึ่งได้รับการพัฒนาเป็นเว็บแอปพลิเคชันรูปแบบ Interactive Dashboard ด้วยเทคโนโลยีสมัยใหม่ เพื่อวิเคราะห์ข้อมูล ค้นหาเทรนด์ และอัปเดตข่าวสารที่ส่งตรงจากผู้ใช้งานจริงในไทย

---

## 🚀 วิธีการเข้าใช้งานและรันเว็บแอปพลิเคชัน (How to Run)

เว็บแอปพลิเคชันพัฒนาขึ้นด้วย **React + Vite** และใช้ **Vanilla CSS** ในการดีไซน์ระดับพรีเมียม (Dark Glassmorphism) โดยมีขั้นตอนการรันใช้งานเครื่องตัวเองง่ายๆ ดังนี้:

### 1. เข้าไปในไดเรกทอรีของแอปพลิเคชัน
เปิด Terminal ขึ้นมาแล้วรันคำสั่ง (ห้ามรัน cd หากพิมพ์แยกใน model แต่รันคำสั่งรวดเดียวใน shell):
```bash
cd web-app
```

### 2. รันเซิร์ฟเวอร์สำหรับพัฒนา (Development Server)
```bash
npm run dev
```
ระบบจะเปิดพอร์ต Local Web Server ขึ้นมา (โดยทั่วไปคือ `http://localhost:5173/` หรือสุ่มพอร์ตถัดไป) สามารถเปิด Browser เข้าใช้งานได้ทันที

## 🌟 ฟีเจอร์เด่นของเว็บแอปพลิเคชัน MEEV

1. **Dashboard & Trends (วิเคราะห์แนวโน้มอย่างมีระดับ):**
   * **สถิติสำคัญ:** แสดงจำนวนรุ่นรถที่วิเคราะห์, อัตราการลดราคาเฉลี่ยต่อรุ่น, ระยะทางวิ่งเฉลี่ย และราคาเฉลี่ยในตลาดแบบเรียลไทม์
   * **Interactive Scatter Plot (กราฟความคุ้มค่า):** กราฟแสดงสัดส่วนระหว่าง "ราคาจำหน่ายปัจจุบัน" และ "ระยะทางวิ่ง" ซึ่งแยกสีตามระบบขับเคลื่อน (RWD/FWD/AWD) สามารถคลิกที่จุดเพื่อเปิดดูข้อมูลรุ่นรถย่อยได้ทันที
   * **Price War Comparison (เปรียบเทียบสงครามราคา):** แผนภูมิเปรียบเทียบราคาเปิดตัวนำเข้า CBU เดิม VS ราคาอัปเดตปัจจุบันหลังเปิดสงครามราคาในไทย แสดงมูลค่าและเปอร์เซ็นต์ราคาดิ่งตัวอย่างชัดเจน
   * **Dynamic Market Insights:** การ์ดดึงข้อมูลวิเคราะห์ข้อมูล เช่น รุ่นรถยนต์ไฟฟ้าที่กินไฟต่ำประหยัดสุด และรุ่นรถที่ราคาลดโหดที่สุด
2. **Model Comparison Grid (เปรียบเทียบและเจาะลึกปัญหารุ่นรถ):**
   * ระบบค้นหาอัจฉริยะ ค้นหาได้ตั้งแต่ชื่อแบรนด์, ชื่อรุ่น ไปจนถึงคีย์เวิร์ดปัญหา (เช่น พิมพ์ว่า "แอร์" หรือ "ช่วงล่าง" เพื่อกรองดูรุ่นรถที่ผู้ใช้ระบุถึงอาการนั้นได้ทันที)
   * ตัวกรองแยกตาม แบรนด์รถยนต์ หรือ ระบบขับเคลื่อน
   * **Side Drawer Detail (สไลด์ข้างรายละเอียด):** คลิกการ์ดรุ่นรถเพื่อเปิดแผงสไลด์ด้านข้าง ดูสเปกของรุ่นย่อยทั้งหมด, ปัญหาประจำรุ่นเด่นๆ และจุดแข็งจากผู้ใช้จริง
3. **News & Update Editor (ระบบจัดการข่าวสารและอัปเดต):**
   * แถบแสดงข่าวสารที่เกี่ยวข้องกับรถแต่ละรุ่น โดยพรีโหลดข้อมูลบทความเจาะลึก 3 เรื่องหลัก (สงครามราคา, วิกฤตประกันภัย คปภ., ปัญหาสถานีชาร์จและการเคลมบริการหลังการขาย)
   * **ระบบเพิ่มข่าวสารใหม่ (Create News Form):** มีแบบฟอร์มให้กรอกเขียนข่าวสารเพิ่มหรืออัปเดตปัญหาใหม่ โดยเลือกรถที่เกี่ยวข้องได้ สามารถเก็บข้อมูลลงระบบฐานข้อมูล LocalStorage ช่วยให้ข้อมูลที่พิมพ์เพิ่มยังคงอยู่เมื่อกด Refresh
   * **ระบบลบข่าวสาร:** สามารถลบข่าวสารที่สร้างขึ้นเองได้อย่างง่ายดาย
